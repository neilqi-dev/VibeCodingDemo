# VIbeCodeDemo

A collection of web development projects and demos.

## Projects

### React Player
A React-based MP3 audio player with waveform visualization.

**Tech Stack:**
- React 19 with TypeScript
- Vite
- Web Audio API

**To run:**
```bash
cd react-player
npm install
npm run dev
```

## Getting Started

1. Navigate to the desired project folder
2. Install dependencies
3. Start the development server
